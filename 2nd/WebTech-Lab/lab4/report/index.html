ผู้ช่วยตัวจิ๋วแต่แจ๋ว ที่นักทำเว็บทุกคนควรรู้ !

Bulma คืออะไร?
Bulma คือ CSS framework ที่ถูกจับมัดรวมกันมาให้เราใช้งาน หมายความว่าหากคุณมีเว็บไซต์ของเดิมอยู่แล้ว คุณสามารถนำเจ้า Bulma ไปใช้ได้เลย โดย Bulma จะกระทบแค่ส่วนของหน้าตาเว็บเท่านั้น ไม่กระทบส่วน logic เพราะตัว Bulma CSS Framework เป็นแค่ css ที่ไม่มี javascrip รวมมาให้ทำให้มันมีขนาดที่เล็ก ซึ่งเหมาะกับคนที่ต้องการเขียน javascript เอง
100% responsive
BuiltWith Flexbox
FREE Opensource on Github
วิธีติดตั้งมี 3 แบบ
1. ผ่าน NPM (แนะนำวิธีนี้)
npm install bulma
2. ผ่าน CDN
https://cdnjs.com/libraries/bulma
3. ดาวน์โหลดไปติดตั้งแอง
https://github.com/jgthms/bulma/tree/master/css
โดย Bulma จะรองรับการย่อ/ขยายหน้าจอได้ 4 แบบคือ
  • mobile จะมีขนาดถึง 768px
  • tablet จะเริ่มที่ขนาด 769px
  • desktop จะเริ่มที่ขนาด 769px
  • widescreen จะเริ่มที่ขนาด 1180px
กริดมหัศจรรย์ (ยืดหดได้)
  • ความสามารถหนึ่งของ Flexbox คือ มันสามารถยืดหดตามพื้นที่ที่มีได้ ไม่ต้องมานั่งกำหนด class แบบ .col-md-6 เหมือนใน Bootstrap / Foundation แบบเก่า ๆ (หรือจะกำหนดมันก็ Fix ความกว้างให้ได้)
คอลัมน์สูงเท่ากันอัตโนมัติ | ข้อมูลจัดกลางอัตโนมัติ | กล่องเต็มจออัตโนมัติ
  • หลายคนต้องเคยเจอปัญหาแบบ “ทำยังไงให้คอลัมน์สูงเท่ากัน” โดยปกติก็จะแก้กันด้วย JavaScript ..แต่ Bulma มีฟีเจอร์นี้มาให้เลย !
  • เรื่องของการจัดกลาง “ถ้าจัดกลางในแนวตั้งทำยังไง” Bulma ใช้ Flexbox ก็แก้ปัญหานี้ได้โดยใส่แค่คลาส is-medium
  • “ทำยังไงให้กล่องสูงเต็มจอ” ซึ่งถ้าเอาแบบชัวร์ ๆ ที่เราทำปกติ คือ ใช้ JavaScript คำนวณความสูงให้ แต่เราแค่ใช้ CSS3 height: 100vh; บรรทัดเดียวจบ
